Fun Projects

  • Opening Ceremony Visitor's Dashboard Project

    An application designed to demonstrate the IoT (Internet of Things) used in the scheduling, quality control, and engineering monitoring process of the Daikin Texas Technology Park. It was used in the grand opening ceremony of the factory with over 5000 visitors and now serves as a tool for the marketing team to explain the IoT of the manufacturing process.

    More technical details:

    • Information from various sources in the manufacturing process are collected and visualized in a user friendly manner, including supply chain material flow, production line real time status, and weekly production, etc.
    • The user interface is integrated with multiple languages, including English, Japanese, Traditional Chines, Simplified Chinese, Korean, and Hindi.
    • This application is deployed to production and widely used as a one-in-all presenting tool by the management.

  • User Management System Project

    The User Management System serves as the gateway of various applications developed by the Systems Engineering Team. It is used internally to control the access rights to data for all the engineers and management staff in the company.

    More technical details:

    • Basic features include new account registration, email verification, login, password reset, and confidential information encoding
    • Permission control based on user groups is enforced to all applications within the User Management System. All user groups are fully configurable from the admin dashboard.
    • This project is written in PHP and is required to be applied to softwares written in C# and python as well. Therefore, APIs are provided for basic features such as identity authentification, user information and permission configuration, etc.
    • This application is currently running in production.

  • Material Supply Status Dashboard Project

    In order to improve the efficiency of the supply chain material scheduling process, a dashboard was developed to visualize the real time status of the material supply and demand in the production lines. This project was initially developed for the material department, but was later requested by other department as well due to its convenience and

    More technical details:

    • Analyze the material consumption data from the production line. Based on the analysis, label each Work Order with different urgency level.
    • The urgency of materials needed for the Work Order are visualized with quantity numbers and different colors to help the Material Department in their supply process.
    • This application is currently used in production.

  • Automated Machine Reports Project

    This project delivers a web application and email notification system now widely used by the engineering department and the manufacturing department in our company to make engineering decisions. Using this tool, the engineers are able to see both real time and historical machine data, which reveals each machine's operational status, error history, as well as production performance.

    More technical details:

    • Collect real time binary data from PLCs with information of each machine's production status and error records. These data are stored in the central database for analysis.
    • Wrote APIs that process the binary raw data and output human readable machine operation reports in the form of downloadable excel files as well as email reports sent out every hour automatically.
    • Design and wrote an engineers-oriented web application to visualize the machine data.
    • Full mobile compability
    • This application is currently running in production and is still under constant upgrades.
