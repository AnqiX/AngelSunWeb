TemplateProfessional

09 Jul 2017

A web template for people to build their own professional websites and make friends with mutual interests. Each page is dynamically generated by pulling page content from MySQL database.

This project is derived from www.angelsunweb.com and published to GitHub in the hope to save people's time in writing their own websites from scratch

Material Department Dashboard

09 Jul 2017

A web application published in the internal company server as a tool to improve material supply efficiency in the production area

It visualizes the material supply status at each production line and provides detailed information of the work flow of each work order; The UI itself is mobile friendly and can be switch to different themes at user's choice

Project from work, not available on GitHub
